# 📂 案例 / Examples

真实的端到端使用案例，展示三省执行智能体集群处理完整任务的全流程。

| # | 案例 | 任务内容 | 涉及部门 | 复杂度 |
|---|------|---------|----------|--------|
| 1 | [竞品分析](competitive-analysis.md) | "分析 CrewAI vs AutoGen vs LangGraph" | 中书→门下→数据分析师+代码架构师+文档编写员 | ⭐⭐⭐ |

每个案例包含：
- 📜 **圣旨**：原始指令
- 📋 **任务编排引擎规划**：任务拆解方案
- 🔍 **安全审查引擎审议**：审核意见（含封驳/返工记录）
- ⚔️ **各部执行结果**：每个部门的产出摘要
- 📮 **最终报告**：任务调度引擎汇总的完整报告

> 💡 这些案例基于真实运行记录整理，任务 ID 和时间戳已脱敏。
